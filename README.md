# TaskFlow

A modern full-stack task management app built with Next.js 16 (App Router), TypeScript, Tailwind CSS, and shadcn/ui.

## Features (So Far)
- App Router with nested layouts
- Responsive navbar with client-side navigation
- Dashboard page
- Tasks page with interactive TaskCard component
- TypeScript everywhere
- Dark theme + modern UI

## Tech Stack
- **Framework**: Next.js 16 (App Router + React 19)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Icons**: Lucide React
- **Deployment**: Vercel

npm run dev
